# waze-scan-closures

Passively scans for user-generated/reported road closures in WME and sends Discord notifications when new closures are reported.

## Regions Configured
* Plains (IA, KS, MN, MO, NE, ND, SD)
* Great Lakes (IN, OH, MI)
* Dominican Republic
* Ireland
* Greece
* Cyprus
* Qatar
* United Arab Emirates
* Nepal
* Ecuador
